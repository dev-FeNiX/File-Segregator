# 📁 File Segregator

A simple yet powerful **Streamlit-based file organizer** that automatically moves files into categorized folders based on their type (videos, images, docs, etc.).

> 🔧 Built with Python and Streamlit  
> 🎨 Designed for productivity & simplicity  
> 🧠 Created by [dev-FeNiX](https://github.com/dev-FeNiX)

---

## 🚀 Features

- 📦 Organizes files into folders like `Videos`, `Images`, `Docs`, `Audios`, `Misc`
- 🧠 Smart file extension detection
- ⚙️ Handles duplicate file names gracefully
- ✅ Clean & interactive Streamlit UI
- 📊 Progress bar and status updates
- 💾 Easily configurable and modular codebase

---

## 📂 File Types Supported

| Category | Extensions |
|----------|------------|
| 🎥 Videos | `mp4`, `mkv` |
| 🎧 Audios | `mp3`, `wav`, `ogg` |
| 🖼️ Images | `jpg`, `jpeg`, `png`, `gif`, `webp`, `svg` |
| 📄 Documents | `pdf`, `doc`, `docx`, `txt`, `ppt`, `pptx`, `xls`, `xlsx`, `csv` |
| 📁 Misc | Everything else |
